Parameter: nAisle¶

The number of aisles in the cabin.

Unit:[ ]
Wiki:http://en.wikipedia.org/wiki/Wide-body_aircraft

Calculation Methods¶

nAisle.calc()¶

Sets the calculation method for the number of Aisles

If the fuselage diameter is fixed calcDfus is chosen else calcPax

Dependencies:
  • Parameter: paxSeats
Sensitivities:
nAisle.calcDfus()¶

Calculates the number of aisles from the diameter of the fuselage Fuselage Diameters taken from Airbus Summer School Slides

Source:Overall Aircraft Preliminary Design, Airbus Summer School, B. Trahmer, 2011
Dependencies:
  • Parameter: dfus
Sensitivities:
nAisle.calcPax()¶

Calculates the number of aisles from the number of passengers.

Source:adapted from DLR-LY-IL Cabin Tool, J. Fuchte, 2011

  • Parameter: paxSeats
