Parameter: refAbreast¶

The reference number of pax abreast describing the layout of the cabin.

In a two-aisle aircraft with seven passengers abreast this may result in a value of 232, meaning that there are two passengers in the outer seatings and three in the middle

Unit:[ ]

Calculation Methods¶

refAbreast.calc()¶

Calculates the reference number of pax abreast from the number of aisles and the number of passengers per Row

Source:adapted from DLR-LY-IL Cabin Tool, J. Fuchte, 2011
