Parameter: refLayout¶

The reference layout for the design mission of this aircraft.

This is a combination of the class layout and the range of the aircraft. This is a parameter that will be exported to generate more detailed cabin layouts using the cabin design tool from J. Fuchte, DLR-LY-IL

Unit:[]

Calculation Methods¶

refLayout.calc()¶

Defines the critical layout for the cabin design from the rangeType and the number of classes. Possible out puts are combinations of the number of classes (1,2,3) and short, medium and long range.

Source:adapted from DLR-LY-IL Cabin Tool, J. Fuchte, 2011
