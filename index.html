Conceptual Aircraft Design¶

Overview¶

VAMPzero is a conceptual aircraft design tool that is quick and easy to use. It is based on well-known handbook methods, object-oriented and easily extendible. The tool enables the user to trace the calculation history and export data to CPACS. It is written by Daniel Boehnke at the German Aerospace Center (DLR e.V.) and licensed under the Apache 2.0 license.

For the quick reader

If you want to try out VAMPzero directly, go to the Getting Started section. You will find information on downloading and installing the code as well as information on how to run some first calculations
